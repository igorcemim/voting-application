# API de Votação
API para gerenciamento de sessões de votação.

## Documentação da API
https://voting-application-igorc.herokuapp.com/docs

## Como executar o projeto

### Pré-requisitos
- JDK 11 (para executar via Gradle)
- Docker e Docker Compose (para executar via Docker)

### Utilizando Docker
```sh
docker-compose up
```

### Utilizando Gradle
No Linux/Mac:
```sh
./gradlew bootRun
```
No Windows:
```sh
./gradlew.bat bootRun
```

### Executar os testes
```sh
./gradlew test
```
